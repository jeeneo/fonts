## Fonts manager

> [!IMPORTANT]
> This app has been put under low-maintenance. I no longer own a Samsung device to guarantee it will work.

This app is very much out of date, be aware of security risks of using it:

1. any apps signed with the same internal signature key can update installed fonts and replace them with something else.
2. my implementation of the shizuku api is fragile, theoretical shell privilege escalation is possible.

I do not have the resources to address these issues.

> Note: the "Samsung Fonts Installer" app on the play store is not an official release, however they have released the sources [here](https://github.com/ahxway/fonts-Onui8/tree/master)

> [!WARNING]
> This does not work on One UI 8.5.
